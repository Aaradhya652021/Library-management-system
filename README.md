# Library-management-system
Library Management System built using Python and SQL to manage books, users, and issue/return operations efficiently with a structured database system.
The Library Management System is a software application developed using Python for the front-end logic and SQL (Structured Query Language) for database management. It is designed to automate and manage the day-to-day operations of a library efficiently.

This system helps in maintaining records of books, students/users, and transactions such as issuing and returning books. It reduces manual work, minimizes errors, and improves overall efficiency.

Objectives
To manage library records digitally
To automate book issuing and returning process
To maintain user and book details in a structured way
To reduce paperwork and save time

Technologies Used
Python → For application logic and user interface (CLI or GUI)
SQL (MySQL / SQLite) → For storing and managing data
Libraries (optional):
tkinter → GUI
sqlite3 / mysql-connector → Database connection

Key Features
Book Management
Add new books
Update book details
Delete books
Search books
User Management
Add new users (students)
View user details
Issue Book
Issue books to users
Store issue date
Return Book
Return books
Calculate fine (if late)
Database Management
Store all records in SQL tables
Maintain relationships between books and users
